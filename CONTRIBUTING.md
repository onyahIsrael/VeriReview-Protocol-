How to contribute to VeriReview

We are excited to build the future of trust in Nigeria on the Avalanche 9000 network. Follow these steps to help us grow:

Our Standards

We write clean code that is easy for everyone to read.

We focus on mobile first designs for our users in Lagos and beyond.

We use Avalanche Interchain Messaging for all trust signals.

The Process

Fork this repository to your own account.

Create a new branch for the feature you are building.

Write your code and test it on the Fuji Testnet.

Submit a Pull Request with a clear explanation of what you changed.

Code of Conduct Be respectful and helpful to every builder in the community. We are all here to win the Games together.
